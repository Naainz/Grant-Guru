Financial Aid Application Assistance

Step-by-step guides for FAFSA and CSS Profile applications and document checklists

FAFSA Application Guide

  1. Create an FSA ID and gather necessary documents.
  2. Complete the FAFSA form at studentaid.gov.
  3. List the schools you are interested in attending.
  4. Answer dependency status questions.
  5. Provide parent financial information (if dependent).
  6. Sign and submit your FAFSA form.
  7. Follow up on your FAFSA submission and make any necessary corrections.

CSS Profile Application Guide

  1. Create a College Board account and gather necessary documents.
  2. Complete the CSS Profile at cssprofile.collegeboard.org.
  3. List the schools you are applying to that require the CSS Profile.
  4. Provide detailed financial information about your family.
  5. Submit your CSS Profile and pay the fee (waivers available for eligible students).
  6. Submit any additional documentation requested by the schools.

Document Checklists

  • Social Security number or Alien Registration number (if applicable).
  • Federal income tax returns, W-2s, and other records of money earned.
  • Bank statements and records of investments.
  • Records of untaxed income (e.g., child support, interest income).
  • A list of schools you are interested in attending.
